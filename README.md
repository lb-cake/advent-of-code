# Project Setup Instructions for UV

## Prerequisites
Make sure you have the following installed:
- [Node.js](https://nodejs.org/en/download/)
- [npm](https://www.npmjs.com/get-npm)
- [uv](https://github.com/user/uv)

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/lb-cake/advent-of-code.git
   cd advent-of-code
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the project:
   ```bash
   npm start
   ```

## Running Tests
To run the tests, use:
```bash
npm test
```
